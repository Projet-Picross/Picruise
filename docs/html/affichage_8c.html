Fonction qui produit le rendu final du jeu Picruise.

Paramètres
windowest une fenêtre générée avec SDL dans laquelle se déroulera l'affichage du jeu Picruise
rendererest le moteur de rendu de la fenêtre, l'espace où nous allons pouvoir faire notre affichage
rreprésente un ensemble de cases, formant une grille, avec lesquelles le joueur peut intéragir
grilleest un pointeur d'entier qui pointe sur les cases de la grille du joueur
mat_cond_lest un pointeur d'entier qui pointe sur les conditions des lignes du picross
mat_cond_cest un pointeur d'entier qui pointe sur les conditions des colonnes du picross
formatreprésente la taille de la matrice
nb_max_cond_lignesest le nombre maximale de condition qu'une ligne du picross puisse obtenir
nb_max_cond_colonnesest le nombre maximale de condition qu'une colonne du picross puisse obtenir
Renvoie
une variable utilisée pour perpétuer la boucle d'intéraction, sa valeur sera 0 quand on arretera le jeu
