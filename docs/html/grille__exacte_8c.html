◆ grille_exacte()

int grille_exacte ( int  mat_cond_l[N][N],
int  mat_cond_c[N][N],
int  combi[N][300][10],
int  grille[N][N] 
)

Génère des grilles possibles pour la résolution du picross à partir des solutions générées par la fonction solve.

Paramètres
mat_cond_lest la matrice contenant les conditions des lignes du picross
mat_cond_cest la matrice contenant les conditions des colonnes du picross
combiest la matrice cubique contenant les solutions générées par la fonction solve
grilleest une matrice contenant des 0 ou des 1 (1 = case noire | 0 = case blanche)
Renvoie
1 quand la fonction est terminée

on teste les combinaisons en rassemblant une ligne de chaque matrices
ex: 0 0 0 0 2 0 0 1 0 1
signifie que l'on prend les lignes d'indice 0 pour les matrices en positions 0,1,2,3,5,6,8 on prend aussi la lignes d'indice 2 de la matrice en position 4
on prend aussi les lignes d'indices 1 des matrices en position 7,9
