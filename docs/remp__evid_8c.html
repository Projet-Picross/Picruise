◆ remp_blanc_noire_c()

int remp_blanc_noire_c ( int  mat_cond[N][N],
int  grille[N][N] 
)

Fonction qui complete le remplissage d'un bloc de cases noires selon la présence de cases blanches sur les colonnes.

Paramètres
mat_condest une matrice contenant les conditions à appliquer sur la deuxième matrice passée en paramètre
grilleest une matrice contenant des 0 ou des 1 (1 = case noire | 0 = case blanche)

Si la case est noire, que les deux cases précédentes existent et sont respectivement blanche et noire et que la condition précedante est rempli, alors les cases suivantes sont forcément noires.

Paramètres
mat_condest une matrice contenant les conditions à appliquer sur la deuxième matrice passée en paramètre
grilleest une matrice contenant des 0 ou des 1 (1 = case noire | 0 = case blanche)
Renvoie
1 si la matrice à été modifiée, 0 sinon

Si la case est noire, que les deux cases précédentes existent et sont respectivement blanche et noire et que la condition précedante est rempli, alors les cases suivantes sont forcément noires.
